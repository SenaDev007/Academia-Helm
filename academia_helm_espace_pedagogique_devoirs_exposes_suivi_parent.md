# Academia Helm — Espace Pédagogique Enseignant : Devoirs, Exercices, Recherches, Exposés et Suivi Parent

## 1. Positionnement

Cette fonctionnalité appartient au module Pédagogie, dans l’Espace Pédagogique Enseignant.

Elle permet à chaque enseignant de donner, suivre, évaluer et communiquer aux parents les activités pédagogiques complémentaires réalisées hors ou en prolongement du cours.

Objectif : permettre au parent de suivre concrètement le travail scolaire de son enfant, au-delà des notes et bulletins.

# 3. Adaptation par niveau scolaire

## 3.1 Maternelle

À la Maternelle, l’approche doit rester qualitative, simple et adaptée à l’âge.

Activités possibles :

- activité d’éveil à refaire à la maison ;
- coloriage ;
- comptine à apprendre ;
- chanson à répéter ;
- mot à reconnaître ;
- image à observer ;
- objet à apporter ;
- geste à pratiquer ;
- histoire à écouter ;
- activité de motricité ;
- activité de langage ;
- activité sensorielle.

Suivi possible :

- activité faite ;
- activité non faite ;
- participation satisfaisante ;
- effort observé ;
- besoin d’accompagnement ;
- parent informé ;
- observation qualitative.

Statuts recommandés :

- Très Satisfaisant ;
- Satisfaisant ;
- Peu Satisfaisant ;
- Non Satisfaisant ;
- Non observé.

## 3.2 Primaire

- récitations ;
- lectures ;
- recherches simples ;
- exposés simples ;
- cahier à compléter ;
- tables à mémoriser ;
- dictée préparée ;
- production écrite ;
- exercices d’anglais ;
- activités de remédiation.

Suivi possible :

- fait ;
- non fait ;
- incomplet ;
- bien fait ;
- à reprendre ;
- récité ;
- non récité ;
- leçon copiée ;
- leçon non copiée ;
- parent notifié.

Au Secondaire, les activités doivent être plus structurées, disciplinaires et orientées autonomie.

# 8. Fonctionnalités avancées à ajouter

L’enseignant peut donner des activités différentes selon :

- élèves faibles ;
- élèves moyens ;
- élèves avancés ;
- élèves absents ;
- élèves ayant besoin de remédiation ;
- élèves bilingues ;
- groupes de travail.

## 8.2 Bibliothèque d’activités

L’enseignant peut enregistrer des modèles :

- modèle d’exercice ;
- modèle de devoir ;
- modèle d’exposé ;
- modèle de recherche ;
- modèle de récitation ;
- modèle de lecture ;
- modèle de remédiation.

## 8.3 Réutilisation

L’enseignant peut :

- dupliquer une activité ;
- réutiliser une activité d’une ancienne période ;
- adapter une activité à une autre classe ;
- partager une activité avec un collègue ;

## 8.4 Pièces jointes

L’enseignant peut ajouter :

- PDF ;
- image ;

- fichier audio ;
- vidéo ;
- lien web ;
- fiche d’exercice ;
- support de cours.

## 8.5 Correction et retour

L’enseignant peut :

- corriger un travail ;
- donner une observation ;
- attribuer une note ;
- donner une appréciation ;
- demander une reprise ;
- valider ;
- rejeter ;
- commenter.

## 8.6 Rappels automatiques

Le système peut envoyer :

- rappel 24h avant échéance ;
- rappel le jour même ;
- rappel après retard ;
- rappel aux parents ;
- rappel aux élèves ;
- rappel à l’enseignant pour correction.

## 8.7 Historique de travail personnel

Pour chaque élève :

- nombre de devoirs donnés ;
- devoirs faits ;
- devoirs non faits ;
- devoirs en retard ;
- récitations réussies ;
- leçons non copiées ;
- exposés présentés ;
- recherches rendues ;
- évolution de l’engagement ;
- alertes pédagogiques.

## 8.8 Indicateur d’engagement élève

Le système peut calculer :

- taux de devoirs faits ;
- taux de devoirs en retard ;
- taux de récitations réussies ;
- taux de leçons copiées ;
- taux de participation aux exposés ;
- score d’engagement pédagogique.

## 8.9 ORION et alertes

ORION peut détecter :

- élève qui ne fait pas régulièrement ses devoirs ;
- élève qui ne copie pas ses leçons ;
- élève souvent en retard dans les rendus ;
- classe avec faible taux de devoirs faits ;
- matière avec beaucoup de non-rendus ;
- parent qui ne consulte jamais les devoirs ;
- risque de décrochage.

## 8.10 Sara AI

Sara AI peut aider l’enseignant à :

- générer des exercices ;
- proposer des thèmes d’exposé ;
- simplifier une consigne ;
- adapter un devoir par niveau ;
- créer une grille d’évaluation ;
- générer des questions de compréhension ;
- reformuler une observation parent ;
- créer des exercices bilingues.

# 12. Conclusion

Cette fonctionnalité enrichit fortement l’Espace Pédagogique Enseignant.

Elle permet de passer d’un simple suivi de notes à un vrai suivi pédagogique quotidien.

La chaîne devient :

1. L’enseignant donne le travail.
2. L’élève réalise ou non le travail.
3. L’enseignant suit et observe.
4. Le parent est informé.
5. ORION détecte les risques.
6. Sara AI assiste la production pédagogique.
7. La direction dispose d’indicateurs fiables.

C’est exactement ce qui renforce la valeur d’Academia Helm face aux solutions classiques.
