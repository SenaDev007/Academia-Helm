# Academia Helm — Module Supplémentaire 8 : EduCast

## 1. Présentation générale

Le module EduCast d’Academia Helm est un espace de diffusion de contenus éducatifs, institutionnels et pédagogiques sous forme de vidéos, audios, podcasts, capsules de cours, annonces, webinaires, émissions scolaires, formations internes et contenus de sensibilisation.

EduCast doit fonctionner comme une plateforme média scolaire intégrée, proche d’un mini YouTube / podcast privé de l’établissement, mais strictement contrôlé par les permissions, la modération et les règles pédagogiques de l’école.

---

## 2. Vision du module

EduCast ne doit pas être un simple dépôt de vidéos.

Il doit devenir :

- une médiathèque éducative ;
- une chaîne officielle de l’école ;
- un canal de formation continue ;
- un espace de communication pédagogique ;
- un support de révision pour les élèves ;
- un outil de valorisation de l’établissement ;
- un canal d’engagement parent-école ;
- un levier de modernisation de l’enseignement.

L’idée est claire : l’école produit, organise et diffuse ses propres contenus éducatifs dans un environnement sécurisé.

---

## 3. Objectif métier

Le module EduCast permet à l’école de :

- publier des vidéos pédagogiques ;
- diffuser des podcasts éducatifs ;
- partager des capsules de révision ;
- organiser des webinaires ;
- créer des chaînes par niveau, classe ou matière ;
- former les enseignants ;
- informer les parents ;
- diffuser les annonces officielles ;
- archiver les événements scolaires ;
- suivre les consultations ;
- mesurer l’engagement ;
- recommander des contenus aux élèves ;
- enrichir l’espace pédagogique.

## 27. Règles métier

- Un contenu destiné aux élèves doit respecter leur niveau et leur âge.
- Un contenu peut nécessiter une validation avant publication.
- Les vidéos contenant des élèves doivent respecter les autorisations parentales.
- Les contenus externes doivent être validés avant diffusion.
- Les parents ne voient que les contenus autorisés.
- Les élèves ne voient que les contenus liés à leur niveau, classe ou groupe.
- Les enseignants ne peuvent gérer que leurs contenus sauf permission spéciale.
- Les statistiques doivent respecter la confidentialité.
- Les contenus sensibles doivent être modérés.
- Les rapports doivent être exportables.
- Les médias doivent être optimisés pour mobile.
- Les contenus doivent pouvoir être archivés.
