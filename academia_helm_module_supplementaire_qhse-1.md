# Academia Helm — Module Supplémentaire 9 : QHSE

## 1. Présentation générale

Le module QHSE d’Academia Helm permet de gérer la Qualité, l’Hygiène, la Sécurité et l’Environnement au sein d’un établissement scolaire.

Il couvre les procédures qualité, les inspections, les incidents, les risques, la santé scolaire, l’hygiène des locaux, la sécurité des élèves, la conformité réglementaire, les plans d’action, les audits internes, les contrôles, les alertes, les rapports et le pilotage directionnel.

## 2. Objectif métier

Le module QHSE permet à l’école de sécuriser les élèves, enseignants et personnels, améliorer l’hygiène des locaux, suivre les incidents et accidents, prévenir les risques, gérer les inspections, documenter les procédures, assurer la conformité interne, suivre les plans d’action, contrôler la qualité des services, gérer les alertes sanitaires ou sécuritaires, produire des rapports QHSE et donner à la direction une vision claire des risques.

## 25. Règles métier

- Tout incident critique doit générer une alerte immédiate.
- Les incidents liés aux élèves doivent être visibles uniquement par les personnes autorisées.
- Les données de santé doivent être protégées.
- Les parents ne doivent voir que les informations concernant leur enfant.
- Une action corrective doit avoir un responsable et une échéance.
- Les actions en retard doivent déclencher une alerte.
- Les audits doivent produire des constats et recommandations.
- Les documents QHSE doivent être versionnés.
- Les contrôles périodiques doivent être planifiables.
- Les risques critiques doivent être remontés à la direction.
- Les rapports doivent être exportables.
- Les accès doivent être journalisés.
- Les informations sensibles doivent respecter la confidentialité.
